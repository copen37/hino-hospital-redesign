# 日野病院サイトリデザイン

- **目的**: 医療法人福寿会 日野病院の旧来サイトを参考に、現代的でスマホ対応されたシングルページを構築。地域に根ざした外来・入院・介護をサポートするコンテンツ構成とした。
- **特徴**:
  - ヒーローに予約CTAと電話リンク、連絡先カードでコンバージョンを近づけるデザイン
  - サービスグリッドにホバーアニメーションを加え、視覚的な情報の読み取りやすさを向上
  - 信頼性を打ち出すセクション（院長理念・認定・チーム医療）と患者リソースリンク
  - フッターにナビ・住所・電話を集約し、アクセス導線を強化
  - モダンなタイポグラフィとカラーで、温かさと信頼感を両立
- **ファイル構成**:
  - `index.html`: ページ本体
  - `styles.css`: レスポンシブなレイアウトとカード・CTAのスタイリング
- **ローカルでの確認手順**:
  1. このディレクトリで静的ホスティング（例: `python -m http.server`）を立ち上げる
  2. ブラウザで `http://localhost:8000` を開き、レスポンシブ表示を確認
- **デプロイ備考**: GitHub Pages や Vercel にアップする際は `index.html` をルートに配置すれば即公開可能。
